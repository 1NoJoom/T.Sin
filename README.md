## 📋 پیش‌نیازهای سیستم

برای تجربه بهترین عملکرد و پایداری، استفاده از مشخصات زیر برای سرور پیشنهاد می‌شود:

* **لوکیشن سرور:** ترجیحاً آلمان 🇩🇪
* **پردازنده (CPU):** حداقل ۲ هسته
* **حافظه رم (RAM):** حداقل ۴ گیگابایت

> **💡 نکته مهم درباره پینگ:** کیفیت ارتباط و پینگ نهایی، ارتباط مستقیمی با اینترنت کاربر دارد. هرچقدر کیفیت اینترنت کاربر با سرور شما سازگاری بهتری داشته باشد، پینگ به نسبت پایین‌تر و اتصال پایدارتر خواهد بود.

---

## 🛠 راهنمای نصب

برای نصب اسکریپت، کافیست با دسترسی روت (Root) وارد ترمینال سرور خود شده و دستور زیر را اجرا کنید:

```bash
sudo bash -c "$(curl -sL "https://raw.githubusercontent.com/1NoJoom/T.Sin/main/install.sh?v=$(date +%s)" | tr -d '\r')"
```
## 🔄 نحوه اجرای مجدد منو
در صورتی که مراحل نصب به پایان رسید یا از منوی برنامه خارج شدید، نیازی به اجرای مجدد دستور نصب نیست. برای دسترسی دوباره به منوی مدیریت ابزار، کافیست کلمه زیر را در ترمینال تایپ و اینتر کنید:

```bash
T.Sin
```
